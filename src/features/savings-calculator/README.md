# C3 Savings Calculator

Interactive savings calculator built around the **C3 Framework** (Cut, Cancel, Combine) for the No Spend Challenge ecosystem.

**Status:** ✅ Complete
**Completed:** December 30, 2024
**Dev Server:** Port 3100
**Live Demo:** https://tweakyourgeek.github.io/nospend-apps-/savings-calculator/

## The C3 Framework

The C3 framework prevents "all-or-nothing deprivation" by offering three approaches to financial wellness:

- **✂️ CUT** - Reduce frequency or eliminate (e.g., skip daily coffee, pack lunch)
- **❌ CANCEL** - Stop paying entirely (e.g., unused subscriptions, memberships)
- **🔗 COMBINE** - Consolidate & share costs (e.g., family plans, carpooling)

## Features

- **Three C3 Sections**: Dedicated calculators for Cut, Cancel, and Combine scenarios
- **Quick Presets**: Common scenarios for each C3 category
  - CUT: Coffee, Lunch, Takeout, Happy Hour
  - CANCEL: Streaming, Gym, Magazines, Premium Apps
  - COMBINE: Family Plans, Cloud Storage, Carpool, Bulk Buying
- **Visual Charts**: Compare all three C3 strategies over 12 months
- **Total Savings Summary**: See combined impact of all three approaches
- **Real-time Calculations**: Results update as you adjust sliders
- **Mobile-First**: Fully responsive design
- **Non-Judgmental**: Warm, encouraging messaging ("This isn't deprivation, it's empowerment")

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The app will run on **http://localhost:3100** (port 3100 to avoid conflicts)

## Usage

1. **Explore each C3 category**: Cut, Cancel, or Combine
2. **Try the presets** or customize with sliders
3. **View your potential savings** per month and per year
4. **See the total impact** of using all three strategies together
5. **Compare visually** using the interactive chart
