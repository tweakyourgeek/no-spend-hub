# Project Status Extraction Prompt Template

Use this prompt in each conversation that contains 30-Day No-Spend Challenge journal work.

---

I need a comprehensive project status extraction for the 30-Day No-Spend Challenge journal work done in this conversation.

**Please organize your response into these categories:**

## 1. COMPLETED DELIVERABLES
List everything that was fully finished in this conversation:
- Specific pages/sections (with page numbers)
- Worksheets or tracking tools
- Daily prompts (which days?)
- Weekly overviews (which weeks?)
- Expense logs or other variations
- Any other content pieces

**For each item, note:**
- Is it currently saved as an artifact/file that can be downloaded?
- Or does it only exist in the chat messages?

## 2. WORK IN PROGRESS
Anything started but not finished:
- What was partially completed?
- What feedback or revisions were discussed but not implemented?
- Any "we'll come back to this" items?

## 3. SHOULD BE ARTIFACTS (but aren't yet)
Content that exists in chat messages but should be compiled into downloadable files:
- Which sections should be combined?
- What needs to be exported for use in other tools?

## 4. STILL MISSING
Based on what we discussed creating but didn't finish:
- What pages/sections are referenced but don't exist yet?
- What was planned for "later" or "next"?

## 5. FRAMEWORK ALIGNMENT
Confirm which framework this content uses:
- Alignment Lab Codex (Lab Loop + Eight Pillars + Four Lenses) with Chasing patterns as optional tools?
- Or older Chasing Series as primary framework?

**Output format:** Use bullet points, be specific about page numbers/section names, and flag anything that needs action.
