## Design Notes

- 4 household types: Singles, Couples, Families, Pet Parents
- 2 vibes each: Irreverent and Regular (still on-brand, just less spicy)
- "Essential?" column replaced with "Need / Gray Area / Want" or "Survival / Sanity / Shiny Thing"
- User decided to use the Regular version (without "Singles" label) as the default in the journal, with all household versions available in the back for users to choose from
- Different household versions planned for separate product launches in Q2
