## Spreadsheet Pack Components

### Core Pack (9 Tabs)
**Source**: Files 7, 17

**Tab 1: README**
- **Type**: Instructions document
- **Content**: How to use each tab, tips, troubleshooting

**Tab 2: Dashboard**
- **Type**: View-only metrics display
- **Features**:
  - Navigation hyperlinks to all tabs
  - Active subscriptions total
  - Monthly freed amount
  - Category pie chart
  - Weekly income vs expenses chart
  - Before vs After visual
  - Freed-Up Money allocation breakdown

**Tab 3: Subscriptions & Autopays**
- **Type**: Data entry table
- **Columns**:
  - Subscription Name
  - Category (dropdown)
  - Monthly Cost
  - Status (Keep/Pause/Cancel dropdown)
  - Billing Cycle (Monthly/Annual/Quarterly dropdown)
  - Last Used
  - Notes

**Tab 4: One-Time Purchases**
- **Type**: Transaction log
- **Columns**:
  - Date
  - Item
  - Category
  - Amount
  - Chasing Pattern (dropdown)
  - Need/Want/Gray Area
  - Notes

**Tab 5: Category Summary**
- **Type**: Auto-calculated aggregation
- **Features**:
  - SUMIF formulas pulling from Tabs 3 & 4
  - Category totals
  - Visual breakdown
  - Top spending categories

**Tab 6: Monthly Overview**
- **Type**: 12-month tracking table
- **Columns per month**:
  - Income
  - Essential Expenses
  - Non-Essential Expenses
  - Net
  - Savings Rate %

**Tab 7: Weekly Overview**
- **Type**: 5-week tracking (covers long months)
- **Columns per week**:
  - Week number
  - Income
  - Expenses
  - No-spend days count
  - Net

**Tab 8: Before/After Comparison**
- **Type**: Auto-calculated comparison
- **Metrics**:
  - Total subscriptions (before/after)
  - Monthly subscription cost (before/after)
  - One-time purchases (before/after)
  - Savings achieved
  - Percentage reduction

**Tab 9: Freed-Up Money Plan**
- **Type**: Allocation planner
- **Columns**:
  - Purpose (e.g., Debt, Savings, Investment)
  - Amount Allocated
  - Priority (1-5)
  - Status (Planned/Active/Complete)
